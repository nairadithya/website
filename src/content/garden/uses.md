# Software
1. Distribution - [Fedora](fedoraproject.org/)
2. Window Manager - [Hyprland](https://hypr.land/) ([GNOME](https://www.gnome.org/) as backup)
3. Notes - [org-mode](https://orgmode.org/) (Google Keep for on the go)
4. Editor - [emacs](https://www.gnu.org/software/emacs/) ([Zed](https://zed.dev) when config breaks)
5. Password Manager - [Bitwarden](https://bitwarden.com/)
6. Terminal - [ghostty](https://github.com/kovidgoyal/kitty)
7. Shell - [zsh](https://www.zsh.org/)
8. Git frontend - [Magit](https://magit.vc/) ([Lazygit](https://github.com/jesseduffield/lazygit) as fallback)
9. System Monitor - [btop](https://github.com/aristocratos/btop)
10. File manager - [dired](https://www.gnu.org/software/emacs/manual/html_node/emacs/Dired.html)
11. PDF Reader - [emacs](https://www.gnu.org/software/emacs/)
12. Browser - [Firefox](https://www.mozilla.org/en-US/firefox/new/)
13. Bookmarking - [Raindrop](https://raindrop.io/)
