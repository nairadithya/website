# Software
1. Distribution - [Arch](http://archlinux.org/)
2. Window Manager - [Hyprland](https://hypr.land/) 
3. Notes - [org-mode](https://orgmode.org/) (WhatsApp self-chat for on the go)
4. Editor - [emacs](https://www.gnu.org/software/emacs/) ([Zed](https://zed.dev) when config breaks)
5. Password Manager - [Bitwarden](https://bitwarden.com/)
6. Terminal - [ghostty](https://ghostty.org/)
7. Shell - [fish](https://fishshell.com/)
8. Git frontend - [Magit](https://magit.vc/) 
9. System Monitor - [btop](https://github.com/aristocratos/btop)
10. File manager - [dired](https://www.gnu.org/software/emacs/manual/html_node/emacs/Dired.html)
11. PDF Reader - [emacs](https://www.gnu.org/software/emacs/) ([Sioyek](https://sioyek.info/) for larger documents)
12. Browser - [Zen Browser](https://zen-browser.app/)
13. Bookmarking - [Raindrop](https://raindrop.io/)
